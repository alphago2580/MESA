# Legacy Code Archive

이 폴더는 MESA v1 (암호화폐 트레이딩 봇) 코드의 아카이브입니다.

## 아카이브 날짜
2026-02-21

## 원본 프로젝트 설명
MESA v1은 암호화폐 자동 트레이딩 봇으로, SMA 전략 기반의 Python 구현체입니다.

## 포함된 파일
- `main.py` - 트레이딩 봇 메인 엔트리포인트
- `src/` - 봇 핵심 코드 (bot.py, exchange.py, strategies/, backtesting/)
- `tests/` - 테스트 코드
- `prototype/` - UI 프로토타입 (Gemini/Opus 생성)
- `example_backtest.py` - 백테스팅 예제
- `requirements.txt` - Python 의존성
- `config.example.json` - 설정 예제

## MESA v2
새로운 MESA v2는 AI 기반 경제 리포트 서비스입니다.
