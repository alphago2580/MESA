# MESA v2 경제 지표 데이터 소스 가이드

> 이 문서는 MESA v2에서 추적하는 핵심 경제 지표, 데이터 소스, 업데이트 주기를 정리합니다.

## 1. 금리 / 통화정책

### 미국 기준금리 (Federal Funds Effective Rate)
| 항목 | 내용 |
|------|------|
| **영문명** | Federal Funds Effective Rate |
| **FRED ID** | `DFF` (일별), `FEDFUNDS` (월별) |
| **업데이트 주기** | 일별 (영업일) |
| **중요도** | ★★★★★ |
| **설명** | 미 연준(Fed)이 설정하는 기준금리. 모든 자산 가격의 기준이 되며, FOMC 회의(연 8회)에서 결정. 금리 인상/인하 사이클은 전체 시장 방향성을 좌우한다. |

## 7. FRED API 무료 사용 가이드

### 7.1 API Key 발급

1. [FRED 웹사이트](https://fred.stlouisfed.org/) 접속
2. 우측 상단 "My Account" > 회원가입 (무료)
3. 로그인 후 [API Keys 페이지](https://fred.stlouisfed.org/docs/api/api_key.html) 접속
4. "Request API Key" 클릭 > 용도 간단히 기술 > 발급

### 7.2 기본 사용법

**엔드포인트**: `https://api.stlouisfed.org/fred/series/observations`

**필수 파라미터**:
| 파라미터 | 설명 | 예시 |
|----------|------|------|
| `series_id` | FRED 시리즈 ID | `DFF`, `UNRATE` |
| `api_key` | 발급받은 API 키 | `abcdef1234567890` |
| `file_type` | 응답 형식 | `json` |

**선택 파라미터**:
| 파라미터 | 설명 | 예시 |
|----------|------|------|
| `observation_start` | 시작일 | `2020-01-01` |
| `observation_end` | 종료일 | `2025-12-31` |
| `frequency` | 집계 주기 변환 | `m` (월별), `q` (분기) |
| `units` | 단위 변환 | `chg` (변화), `pch` (퍼센트변화), `pc1` (전년대비%) |

**요청 예시**:
```
GET https://api.stlouisfed.org/fred/series/observations?series_id=DFF&api_key=YOUR_KEY&file_type=json&observation_start=2024-01-01
```

**응답 예시** (JSON):
```json
{
  "observations": [
    { "date": "2024-01-01", "value": "5.33" },
    { "date": "2024-01-02", "value": "5.33" }
  ]
}
```

### 7.3 Rate Limit 및 주의사항

- **요청 제한**: 분당 120회 (초당 2회 정도)
- **일일 제한**: 없음 (공식적으로 명시되지 않음)
- **데이터 지연**: 대부분 발표일 당일 또는 익영업일 업데이트
- **비용**: 완전 무료 (상업적 사용 포함)
- **라이선스**: 대부분의 데이터는 공개 도메인 (public domain)

### 7.4 유용한 FRED 시리즈 검색
